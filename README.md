# Simple-Paint
study project
В проекте можно рисовать с помощью круга. Рисовать можно как и с помощью мышки, так и с помощью клавиатуры, чтобы линии были ровные.
Можно менять цвет круга если кликнуть на круг и нажать букву на клавиатуре. Также можно сделать заливку объекта если он замкнутый.
Поменять на разные цвета:
Зеленый - 'g'
Красный - 's'
Синий - 'f'
Фиолетовый - 'd'
Желтый - 'a'
Сделать заливку:
Начать - 'w'
Закончить - 'e'
Если хотите поделится этим проектом перейдите по этой ссылке https://learn.algoritmika.org/share/59186308
